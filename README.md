# superman
A repository for data for the paper "An extremely luminous flare recorded from a supermassive black hole" (https://www.nature.com/articles/s41550-025-02699-0)

   * phot/ contains the photometry from ZTF, CRTS, ATLAS, and WISE
   * spec/ contains the DBSP and LRIS spectra
   * KCWI data is available on request (too large for Github)
